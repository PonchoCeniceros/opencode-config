# Reglas globales para desarrollo web

## Stack Principal
- **Frontend**: Next.js + React Native/Expo
- **Backend**: NestJS + Express.js
- **DB**: Firebase, MongoDB, Supabase
- **Lenguaje**: TypeScript

## Convenciones de Código
- TypeScript strict mode
- Imports: `import` antes que `import type`
- Tests con Jest/Vitest
- Commits convencionales

## Skills Disponibles
- **nestjs-expert** → Arquitectura NestJS, DI, testing
- **express-expert** → Express.js, middleware, routing
- **typescript-expert** → Tipos, generics, configuración TS
- **web-design-guidelines** → UI/UX review
- **nextjs-expert** → Next.js App Router
- **supabase-expert** → RLS, Auth, Edge Functions
- **expo-expert** → React Native/Expo

## Agentes
- **mobile-dev** → Experto en React Native y Expo
- **web-review** → Revisión de código full-stack

## Patrones Preferidos
- React: Functional components + Hooks
- Backend: Module-first (Nest) / Controller-service (Express)
- Estado: React Query
